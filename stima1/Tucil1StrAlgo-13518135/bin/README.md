# Tucil1-Stima -- Penyelesaian Persoalan Convex Hull dengan Algoritma Brute Force
Program ini adalah contoh pengaplikasian strategi algoritma Brute Force untuk menyelesaikan persoalan Convex Hull

## Instalasi Program
Program ini dibuat menggunakan bahasa pemrograman java oleh karena itu dibutuhkan compiler Java SE versi 8 atau lebih tinggi. Compiler dapat diunduh melalui https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

### Cara Kompilasi Kode Program
Untuk mengompilasi program, buka terminal komputer Anda dan pergi ke directory di mana program disimpan. Kemudian silakan ketikkan perintah seperti di bawah ini:

```
C:\Users\<user>\Documents\Tucil1StrAlgo-13518135>
# cd bin
C:\Users\<user>\Documents\Tucil1StrAlgo-13518135\bin>
# javac -d ConvexHull.java
```
Setelah perintah di atas diketikkan dan tidak muncul pesan error, maka program berhasil di kompilasi

### Cara Menjalankan Program #1
Untuk menjalankan program, cukup double-click pada file ConvexHull.exe dan program akan berjalan

### Cara Menjalankan Program #2
Setelah program berhasil dikompilasi, maka program dapat dijalankan. Untuk menjalankan programnya, pada terminal dan dari directory bin silakan jalankan programnya. Berikut adalah instruksi untuk menjalankan programnya :

```
C:\Users\<user>\Documents\Tucil1StrAlgo-13518135\bin>
# java ConvexHull.java

[ CONVEX HULL GENERATOR ]
Silakan masukan jumlah node:
```
Jika program berhasil dijalankan, maka akan muncul menu utama program seperti di atas.

## Cara Menggunakan Program
Ketika program dijalankan, maka Anda hanya perlu mengetikkan jumlah titik yang akan dibuat. Lalu program akan membuat sejumlah titik secara acak, mengkalkulasikan convex hullnya, dan mengilustrasikannya.

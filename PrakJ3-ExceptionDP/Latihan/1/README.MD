Pada soal ini Anda diminta untuk membuat kelas Car yang merupakan representasi dari objek mobil dengan menerapkan konsep Exceptions. Untuk membuat kelas ini, Anda harus membuat 4 file java, yaitu Car.java, FuelException.java, InvalidInputCarException.java, dan TheCarIsOnException.java dengan rincian sebagai berikut.

```
Car.java

FuelException.java

    Memiliki sebuah member String bernama fuel.
    Memiliki konstruktor dengan satu parameter bertipe String.
    Method getMessage() yang mengembalikan pesan "Bensin jenis <fuel> tidak sesuai". 

InvalidInputCarException.java

    Memiliki sebuah member integer bernama input.
    Memiliki konstruktor dengan satu parameter bertipe integer.
    Method getMessage() yang mengembalikan pesan "Input <input> diluar range". 

TheCarIsOnException.java

    Memiliki konstruktor tanpa parameter yang tidak melakukan apapun.
    Method getMessage() yang mengembalikan pesan "Mobil sudah menyala".  
```

Kumpulkanlah [Car.java, FuelException.java, InvalidInputCarException.java, dan TheCarIsOnException.java] yang dimasukkan menjadi satu file zip.
